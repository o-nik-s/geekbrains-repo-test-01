Урок 2. Базовые операции
Clone - выполняется единоразово
Commit - сохраняет изменения в локальном репозитории
Push - отправляет изменения на удаленный репозиторий
Put - позволяет получить изменения из удаленного репозитория и скачать их на компьютер

Урок 3. Откат изменений кода
Discard - позволяет не оптравлять в репозиторий те изменения, которые нам не нравятся
Recert - позволяет откатить существующие Commit-ы
Reset - выкинули из репозитория целую пачку commit-ов

Урок 4. Работа с ветками
Add branch,,, - добавление ветки
Merge - слияние веток

Урок 5. Конфликты
Abort merge - откат слияния
Двойным нажатием открываем Conflict Solver
И дальше самостоятельно принимаем решение, как должен выглядить результирующий код
